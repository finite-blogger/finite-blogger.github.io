Getting Started With CSS

CSS stands for Cascading Style Sheets, used for structuring and styling websites. You can add CSS to HTML in three ways.

The first way is to use the style=" " attribute in the opening tag of your element and write your CSS code inside the quotations.

< p style="color: blue; background: red; " > Here goes some text </p>

The next two ways are pretty much similar. The only difference is one is used within the HTML file while the other is used in a separate file.

In your HTML file, in the head tag, put a style tag.

<head>
<title>page title <title/>
<style>

p{
color:blue;
background:red;
}
<style/>
<head/>

If want to write your CSS code in a separate file, you can link the HTML file to the CSS file using the link tag.

<head>
<title>page title <title/>
<style>

<link rel="stylesheet" href="style.css">
<style/>
<head/>
