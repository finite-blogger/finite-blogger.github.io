How to get started with HTML

In HTML, code is written in tags. Each element in HTML is wrapped in an opening tag (<element>) and a closing tag (</element>) though some do not have closing tags like the image tag.

There are two types of elements in HTML, block-level elements and inline elements.
Block-level elements are elements that always start on a new line for example <div>, <p> and <table>.
Inline elements are elements that can be place within the line for example <button>, <span> and <input>.

In your index.html file, type the following.

<!DOCTYPE html>
<html lang="en">
<head>
<title> page title </title>
</head>
<body>
<h1>My Website</h1>
<!-- page content -->

</body>
</html>

Break down

<!DOCTYPE html>: specifies the document type which in this case is HTML.
<html></html>: wraps the whole webpage.
<head></head>: contains the metadata of the webpage
<title></title>: The title of your webpage
<body></body>: contains the main content of the webpage.
<h1></h1>: the heading
<!-- -->: comment, it's not run by the browser but it can be used to explain your code.
