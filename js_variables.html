JavaScript Variables

A variable is simply an element or a group of elements in JavaScript. Variables can be created in different ways. You can either select elements from the HTML code or create them via JavaScript.

Creating variables, we use the const keyword and let which was var in older versions of JavaScript.
Const is used for variables whose values are constant while let and var are used for variables whose values can be changed.
Properties like getElementById() and getElementsByClassName() are used to select elements from the HTML code.

<body>
<div id="text">
</div>
<script>

let text = document.getElementById("text");
text.innerHTML = `<p>Hello World!<p>`;

<script/>
<body/>

The code above writes a text that says "Hello World!" in the div with the"id" of "text".
