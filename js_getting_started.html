Getting Started With JavaScript

JavaScript, abbreviated as JS, is a lightweight programming language used to create interactive webpages to interact with visitors. It is a multi-paradigm, object oriented and functional scripting language. It involves use of operators and functions.

First of all, to add JavaScript to a webpage, there are two methods.
One is to write the JS code in the script tag at the bottom of the body in the HTML file.
The other is to write in a separate file with a .js extension.

Within the HTML file

<body>

<script>

alert("Hello World!");
<script/>
<body/>

Using a .js file

<body>

<script src="script.js" >

<script/>
<body/>
