How to add images to an HTML website

Images are a very important part of a website. They make your website more visually appealing to visitors.

Images can be added to HTML code using the <img>tag and the src=" "

First make sure you have your image in an images folder and then use the src=" " attribute to get the file path of the image.

< img
src = "folder/image_name.file_extension"
>

For images, the supported file extensions are jpg, jpeg and png.

The images can be styled with CSS for example you can change the size, border, opacity and so on.

img {
height: 500px;
width: 300px;
border-radius: 20px;
}
