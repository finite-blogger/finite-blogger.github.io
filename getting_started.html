How to get started with web development

To create a static website you are going to need some basic knowledge of HTML, CSS and JavaScript. You need a code editor like VsCode for computers and if you're using a mobile phone, you can use TrebEdit, Spck or Acode.

HTML
It stands for Hypertext Markup Language. Create a file with a .html extension in your code editor. For example index.html. This file is going to contain the components of the webpage. A website can have either on or many webpages. To learn more on getting started with HTML, you can check out this post.

CSS
It stands for Cascading Style Sheets. It is used for adding styles to the components of a webpage for example background color, font size, font family and so on. It's file always has a .css extension. For instance style.css. And it can be linked to the HTML file using the link tag. To learn more about getting started with CSS, check out this post.

JAVASCRIPT
This is the major programming language for static websites. It is used to add interactivity to a webpage for example click events, loops, functions and so on. To add it, you have to create a .js file and link it to the HTML file using the script tag. And here is the post.
